Detailed Description

ChassisDefn.h.

Create a chassis from an XML definition. CHASSIS_TYPE (ChassisFactory.h) determines the type of chassis to create. WheelBase is the front to back distance between the wheel centers. Track is the left to right distance between the wheels.
